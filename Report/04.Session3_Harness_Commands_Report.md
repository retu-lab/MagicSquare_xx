## 4) 검증·결과

- `pytest --collect-only` → **0 tests collected** (의도된 골격 상태)
- 함정 격자 탐색: **1~16 중복 없이 행·열 8선 합=34이면 대각선도 34** → “행·열=34, 대각선≠34” 데이터 함정은 존재하지 않음
- Mom Test 함정은 **`incomplete`(대각선 미검증)** 시나리오로 Test Loop에 반영 (`TRAP_ROWS_COLS_OK`)
