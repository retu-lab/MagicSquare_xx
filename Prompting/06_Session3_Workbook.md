# 세션 3 — 워크북 (Mom Test → 8계층)

## 프롬프트

```
Mom Test 결과:
- 페르소나: [...]
- 진짜 문제 (한 문장): [...]
- Mom Test 증거 3줄: [...]
MagicSquare_xx 세션 3 워크북을 채워줘:
1) 주제 한 문장 (Mom Test 기반, 솔루션 최소화)
2) R-G-I-O (Role/Goal/Input/Output)
3) 성공 기준 3개 (Mom Test 증거와 연결)
4) 표면 문제 — 이번 프로젝트에서 하지 않을 것
8계층 중 이번 세션에서 만드는 것만: Rule, Command, (Skill), Test Loop
```

## 프롬프트 — 보고서·프롬프트 저장

```
Report 폴더와 Prompt 폴더에 보고서와 프롬프트 저장!!!
```

## 작성 기준

| 항목 | 작성 기준 |
|------|-----------|
| **주제 한 문장** | Mom Test 진짜 문제 기반, 솔루션·기능명 최소화 |
| **R-G-I-O** | Role / Goal / Input / Output 표 |
| **성공 기준 3개** | Mom Test 증거 3줄과 1:1 연결 |
| **표면 문제** | 이번 프로젝트에서 **하지 않을** 솔루션·범위 |
| **8계층** | Rule, Command, (Skill), Test Loop만 — 나머지 계층 명시적 제외 |
