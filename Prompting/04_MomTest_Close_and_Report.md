# 인터뷰 종료 & 보고서보내기

## 프롬프트 — 보고서보내기

```
Report 폴더에 보고서를보내줘
```

## 프롬프트 — 프롬프트 저장 (본 파일 생성용)

```
Prompt 폴더에 프롬프트도 저장해줘
```

## 보고서 출력 규격

**경로:** `Report/01.MomTest_Report.md`

**포함 섹션:**

1. 페르소나
2. 인터뷰 답변 (과거 사실)
3. 표면 문제 (금지 — 솔루션 섞인 문장)
4. 진짜 문제 (한 문장)
5. Mom Test 증거 3줄
6. 멘토 코멘트
7. 인터뷰 질문 로그 (요약 표)

## 멘토 종료 코멘트 가이드

- STEP 2로 넘길 때 **솔루션 제안 대신** 진짜 문제(행동·습관)를 전제로 안내
- 증거 3줄이 진짜 문제 문장과 **논리적으로 연결**되는지 확인
