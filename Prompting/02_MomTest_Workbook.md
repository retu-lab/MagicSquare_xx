# Mom Test 워크북

## 프롬프트 (템플릿 요청)

```
## Mom Test 워크북 — MagicSquare_XX
### 페르소나: 
### 인터뷰 답변 (과거 사실):
### 표면 문제 (금지 — 솔루션 섞인 문장):
### 진짜 문제 (한 문장):
```

## 프롬프트 (인터뷰 종료 후 워크북 정리)

```
이걸로 momtest를 종료해줘
```

## 기대 출력 형식

멘토(AI)는 인터뷰에서 수집한 **과거 사실만**으로 아래 4항목을 채운다.

| 항목 | 작성 기준 |
|------|-----------|
| **페르소나** | 인터뷰한 **한 사람**의 역할·상황 |
| **인터뷰 답변** | 날짜·시간·횟수·행동 등 **구체적 사실** bullet |
| **표면 문제** | 솔루션·기능·앱 이름이 섞인 문장 (~~취소선~~, 기록만) |
| **진짜 문제** | 제품 언급 없이 **한 문장** |
