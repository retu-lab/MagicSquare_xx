# Mom Test 증거 3줄

## 프롬프트

```
Mom Test 증거 3줄도 알려줘
```

## 작성 기준

각 줄은 반드시:

- **과거 사실** (이미 일어난 일)
- **구체적** (시간, 행동, 순서)
- **의견·가설·솔루션 없음** ("~하면 좋겠다" 금지)

## 이번 프로젝트 결과 예시

1. 빈칸 2개를 넣은 뒤 행·열 합은 34로 맞아 보였지만, 대각선 하나를 빼먹어 약 20분을 허맸다.
2. 그 20분 동안 대각선은 직접 더해 보지 않았고, 문제 원인을 계산 실수로만 의심하며 찾았다.
3. 대각선을 처음 더해 본 시점은 20분이 지난 뒤였고, 그때서야 계산이 이상하다는 걸 알았다.
